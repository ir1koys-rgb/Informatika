Pengenalan Bahasa C

Apa itu Bahasa C?

Bahasa C adalah bahasa pemrograman yang dikembangkan oleh Dennis Ritchie pada tahun 1972 di Bell Labs. C sering disebut sebagai "induk" dari banyak bahasa pemrograman modern seperti C++, Java, Python, dan PHP.

💡 Fakta Menarik: Sistem operasi UNIX dan Linux ditulis menggunakan bahasa C!

Karakteristik Utama

Karakteristik Penjelasan
Procedural Program disusun berdasarkan prosedur atau fungsi
Low-Level Memiliki akses ke memori (pointer) seperti assembly
Portable Kode bisa dikompilasi di berbagai platform
Efisien Menghasilkan kode mesin yang cepat dan hemat memori
Case-Sensitive Huruf besar dan kecil dibedakan

Struktur Dasar Program C

#include <stdio.h> // Memanggil library int main() { // Fungsi utama // Deklarasi variabel int angka = 10; // Statement printf("Hello, World!\n"); return 0; // Mengakhiri program }

Proses Kompilasi

1. Preprocessing 2. Compiling 3. Assembling 4. Linking

Tipe Data Ukuran Deskripsi Format

Kontrol Alur Program

Percabangan (If-Else)

📘 Materi: Percabangan IF–ELSE

1. Pengertian IF–ELSE

Percabangan IF–ELSE adalah struktur kontrol dalam pemrograman yang digunakan untuk mengambil keputusan berdasarkan suatu kondisi.

➡️ Program akan:

  • Menjalankan perintah jika kondisi benar (true)
  • Menjalankan perintah lain jika kondisi salah (false)

2. Konsep Dasar

Contoh dalam kehidupan sehari-hari:

"Jika nilai bagus, maka lulus. Jika tidak, maka tidak lulus."

3. Struktur Kontrol Keputusan

🔹 if:

if (kondisi) {
    // perintah yang dijalankan jika kondisi benar
}

3.Struktur Kontrol Keputusan

🔹 if-else:

if (kondisi) {
    // dijalankan jika kondisi benar
} else {
    // dijalankan jika kondisi salah
}

4. Alur Kerja IF–ELSE

  1. Program mengecek kondisi
  2. Jika kondisi true → jalankan blok IF
  3. Jika kondisi false → jalankan blok ELSE

5. Contoh Sederhana (Pseudocode)

DECLARE nilai AS INTEGER
SET nilai = 80

IF nilai >= 75 THEN
    PRINT "Lulus"
ELSE
    PRINT "Tidak Lulus"
END IF

📌 Penjelasan:

  • Jika nilai ≥ 75 → "Lulus"
  • Jika tidak → "Tidak Lulus"

6. Contoh dalam Bahasa C

#include <stdio.h>

int main() {
    int nilai = 80;

    if (nilai >= 75) {
        printf("Lulus\n");
    } else {
        printf("Tidak Lulus\n");
    }

    return 0;
}

Switch Case

📘 Materi: Percabangan Switch Case

1. Pengertian Switch Case

Switch Case adalah struktur percabangan yang digunakan untuk memilih satu aksi dari banyak pilihan berdasarkan nilai tertentu.

➡️ Digunakan ketika:

  • Memiliki banyak pilihan kondisi
  • Nilai yang dibandingkan bersifat tetap (misal: 1, 2, 3)

2. Konsep Dasar

Contoh dalam kehidupan sehari-hari:

"Jika angka 1 maka Senin, 2 maka Selasa, dan seterusnya."

3. Struktur Penulisan

🔹 Bentuk Umum:

SWITCH (variabel)
    CASE nilai1:
        perintah
        BREAK
    CASE nilai2:
        perintah
        BREAK
    DEFAULT:
        perintah
END SWITCH

4. Penjelasan Komponen

  • SWITCH → memilih variabel yang dicek
  • CASE → nilai pembanding
  • BREAK → menghentikan proses
  • DEFAULT → dijalankan jika tidak ada yang cocok

5. Contoh Sederhana (Pseudocode)

DECLARE hari AS INTEGER
SET hari = 2

SWITCH (hari)
    CASE 1:
        PRINT "Senin"
        BREAK
    CASE 2:
        PRINT "Selasa"
        BREAK
    CASE 3:
        PRINT "Rabu"
        BREAK
    DEFAULT:
        PRINT "Hari tidak valid"
END SWITCH

📌 Penjelasan:

  • Jika hari = 2 → "Selasa"
  • Jika tidak cocok → "Hari tidak valid"

6. Contoh dalam Bahasa C

#include <stdio.h>

int main() {
    int hari = 2;

    switch(hari) {
        case 1:
            printf("Senin\n");
            break;
        case 2:
            printf("Selasa\n");
            break;
        case 3:
            printf("Rabu\n");
            break;
        default:
            printf("Hari tidak valid\n");
    }

    return 0;
}

7. Perbedaan dengan IF–ELSE

  • IF–ELSE → untuk kondisi kompleks (>, <, dll)
  • Switch Case → untuk nilai pasti
  • Switch lebih rapi untuk banyak pilihan

8. Contoh Kasus Nyata

Menentukan Menu Makanan

DECLARE pilihan AS INTEGER
SET pilihan = 1

SWITCH (pilihan)
    CASE 1:
        PRINT "Nasi Goreng"
        BREAK
    CASE 2:
        PRINT "Mie Ayam"
        BREAK
    CASE 3:
        PRINT "Bakso"
        BREAK
    DEFAULT:
        PRINT "Menu tidak tersedia"
END SWITCH

9. Kesalahan yang Sering Terjadi

  • Lupa menggunakan BREAK
  • Nilai CASE tidak sesuai
  • Tidak menggunakan DEFAULT

10. Kesimpulan

  • Switch Case digunakan untuk banyak pilihan nilai tetap
  • Membuat kode lebih rapi dan mudah dibaca
  • Cocok untuk kondisi sederhana dan terstruktur

Perulangan (Loops)

📘 Materi: Perulangan (Looping)

1. Pengertian Perulangan

Perulangan (Looping) adalah proses menjalankan perintah secara berulang-ulang selama kondisi tertentu masih terpenuhi.

➡️ Tujuan penggunaan:

  • Menghemat penulisan kode
  • Menghindari pengulangan manual

2. Konsep Dasar

Contoh dalam kehidupan sehari-hari:

"Ulangi langkah berjalan sampai tujuan tercapai."

3. Jenis-Jenis Perulangan

  • FOR
  • WHILE
  • DO WHILE

4. Perulangan FOR

Digunakan jika jumlah perulangan sudah diketahui.

🔹 Pseudocode:

FOR i = 1 TO 5 DO
    PRINT i
END FOR

🔹 Contoh C:

#include <stdio.h>

int main() {
    int i;

    for(i = 1; i <= 5; i++) {
         printf("Assalamualaikum Siswa Smaring\n");
    }

    return 0;
}

5. Perulangan WHILE

Digunakan jika jumlah perulangan belum diketahui pasti.

🔹 Pseudocode:

DECLARE i AS INTEGER
SET i = 1

WHILE i <= 5 DO
    PRINT i
    SET i = i + 1
END WHILE

🔹 Contoh C:

#include <stdio.h>

int main() {
    int i = 1;

    while(i <= 5) {
         printf("Assalamualaikum Siswa Smaring\n");
        i++;
    }

    return 0;
}

6. Perulangan DO WHILE

Perulangan yang dijalankan minimal satu kali.

🔹 Pseudocode:

DECLARE i AS INTEGER
SET i = 1

DO
    PRINT i
    SET i = i + 1
WHILE i <= 5

🔹 Contoh C:

#include <stdio.h>

int main() {
    int i = 1;

    do {
         printf("Assalamualaikum Siswa Smaring\n");
        i++;
    } while(i <= 5);

    return 0;
}

7. Perbedaan FOR, WHILE, DO WHILE

  • FOR → jumlah perulangan diketahui
  • WHILE → kondisi dicek di awal
  • DO WHILE → kondisi dicek di akhir

8. Contoh Kasus Nyata

Mencetak angka genap dari 1 sampai 10

FOR i = 1 TO 10 DO
    IF i MOD 2 = 0 THEN
        PRINT i
    END IF
END FOR

9. Kesalahan yang Sering Terjadi

  • Infinite loop (perulangan tidak berhenti)
  • Salah kondisi
  • Lupa increment atau decrement

10. Kesimpulan

  • Perulangan digunakan untuk menjalankan perintah berulang
  • Terdapat FOR, WHILE, dan DO WHILE
  • Sangat penting dalam pemrograman

Array dan String

📘 Materi: Array dan String

1. Pengertian Array

Array adalah struktur data yang digunakan untuk menyimpan banyak data dalam satu variabel dengan tipe yang sama.

  • Menyimpan banyak nilai sekaligus
  • Mempermudah pengolahan data

2. Ilustrasi Array

Nilai = [80, 75, 90, 85]

Indeks:   0   1   2   3

3. Deklarasi Array (Pseudocode)

DECLARE nilai[4] AS INTEGER

SET nilai[0] = 80
SET nilai[1] = 75
SET nilai[2] = 90
SET nilai[3] = 85

4. Contoh Array dalam Bahasa C

#include <stdio.h>

int main() {
    int nilai[4] = {80, 75, 90, 85};

    printf("%d\n", nilai[0]);

    return 0;
}

5. Perulangan pada Array

FOR i = 0 TO 3 DO
    PRINT nilai[i]
END FOR

6. Pengertian String

String adalah kumpulan karakter (huruf, angka, simbol) yang dianggap sebagai satu data.

➡️ String sebenarnya adalah array dari karakter.

7. Ilustrasi String

Nama = "IRWAN"

Indeks:  0   1   2   3   4
         I   R   W   A   N

8. Contoh String dalam Bahasa C

#include <stdio.h>

int main() {
    char nama[] = "IRWAN";

    printf("%s\n", nama);

    return 0;
}

9. Mengakses Karakter String

PRINT nama[0]   // I
PRINT nama[2]   // W

10. Contoh Kasus String

Menghitung jumlah huruf

DECLARE nama AS STRING
SET nama = "IRWAN"

SET i = 0

WHILE nama[i] TIDAK KOSONG DO
    i = i + 1
END WHILE

PRINT i

11. Perbedaan Array dan String

  • Array → menyimpan banyak data (angka/karakter)
  • String → khusus menyimpan teks
  • String adalah bagian dari array

12. Kesalahan yang Sering Terjadi

  • Salah indeks (out of bounds)
  • Lupa bahwa indeks dimulai dari 0
  • Salah memahami string sebagai tipe berbeda

13. Kesimpulan

  • Array menyimpan banyak data dalam satu variabel
  • String adalah array karakter
  • Keduanya sangat penting dalam pemrograman

Pointer (Konsep Penting!)

⚠️ Pointer adalah konsep tersulit tapi paling penting di C!
Pointer adalah variabel yang menyimpan alamat memori dari variabel lain.

Simbol Pointer

Simbol Nama Fungsi Contoh
& Address-of Mengambil alamat memori &x
* Dereference Mengakses nilai di alamat *ptr

Contoh Dasar Pointer

int x = 10; int *ptr = &x; // ptr menyimpan alamat x printf("%d", x); // Output: 10 (nilai) printf("%p", ptr); // Output: 0x7ffee... (alamat) printf("%d", *ptr); // Output: 10 (nilai di alamat ptr) // Mengubah nilai x melalui pointer *ptr = 20; // Sekarang x = 20

Fungsi dalam C

Apa itu Fungsi?

Fungsi adalah blok kode yang melakukan tugas tertentu dan dapat digunakan kembali. Fungsi membantu membuat kode lebih terorganisir dan mudah dipelihara.

Struktur Fungsi

// Deklarasi (Prototipe) int tambah(int a, int b); int main() { int hasil = tambah(5, 3); printf("%d", hasil); // Output: 8 return 0; } // Definisi int tambah(int a, int b) { return a + b; }

Void vs Return Value

Perbedaan utama terletak pada apakah fungsi mengembalikan nilai atau tidak.

void (Tidak kembali)
void sapa() { printf("Halo!"); }
int (Kembali nilai)
int tambah() { return 5; }

Scope Variabel (Lokal vs Global)

  • Variabel Lokal: Hanya bisa diakses di dalam fungsi tempat ia dideklarasikan.
  • Variabel Global: Dideklarasikan di luar fungsi, bisa diakses oleh semua fungsi.
⚠️ Best Practice: Hindari penggunaan variabel global berlebihan karena dapat membuat kode sulit dilacak (debugging).

Fungsi Rekursif

Fungsi yang memanggil dirinya sendiri. Contoh klasik: Faktorial.

int faktorial(int n) { if (n == 1) return 1; return n * faktorial(n - 1); } // faktorial(5) = 5 * 4 * 3 * 2 * 1 = 120
