📖 Pengenalan Bahasa C

Apa itu Bahasa C?

Bahasa C adalah bahasa pemrograman yang dikembangkan oleh Dennis Ritchie pada tahun 1972 di Bell Labs. C sering disebut sebagai "induk" dari banyak bahasa pemrograman modern seperti C++, Java, Python, dan PHP.

💡 Fakta Menarik: Sistem operasi UNIX dan Linux ditulis menggunakan bahasa C!

Karakteristik Utama

Karakteristik Penjelasan
Procedural Program disusun berdasarkan prosedur atau fungsi
Low-Level Memiliki akses ke memori (pointer) seperti assembly
Portable Kode bisa dikompilasi di berbagai platform
Efisien Menghasilkan kode mesin yang cepat dan hemat memori
Case-Sensitive Huruf besar dan kecil dibedakan

Struktur Dasar Program C

#include <stdio.h> // Memanggil library int main() { // Fungsi utama // Deklarasi variabel int angka = 10; // Statement printf("Hello, World!\n"); return 0; // Mengakhiri program }

Proses Kompilasi

1. Preprocessing 2. Compiling 3. Assembling 4. Linking

🔤 Dasar-Dasar Bahasa C

Tipe Data Primitif

Tipe Data Ukuran Deskripsi Format
int 4 byte Bilangan bulat %d
float 4 byte Desimal (presisi tunggal) %f
double 8 byte Desimal (presisi ganda) %lf
char 1 byte Karakter tunggal %c

Variabel

int umur = 20; float harga = 15.5; char inisial = 'A'; // Tanda kutip satu
⚠️ Aturan Penamaan Variabel:
  • Harus dimulai dengan huruf atau underscore (_)
  • Tidak boleh mengandung spasi
  • Tidak boleh menggunakan kata kunci (int, return, if, dll)

Input dan Output

#include <stdio.h> int main() { int angka; // Output ke layar printf("Masukkan angka: "); // Input dari user (perhatikan tanda &) scanf("%d", &angka); printf("Anda memasukkan: %d", angka); return 0; }

Operator Fungsi Contoh Hasil

🎛️ Kontrol Alur Program

Percabangan (If-Else)

int nilai = 75; if (nilai >= 70) { printf("Lulus\n"); } else if (nilai >= 50) { printf("Remedial\n"); } else { printf("Gagal\n"); }

Switch Case

int hari = 3; switch (hari) { case 1: printf("Senin"); break; case 2: printf("Selasa"); break; case 3: printf("Rabu"); break; default: printf("Hari tidak valid"); }

Perulangan (Loops)

// For Loop for (int i = 0; i < 5; i++) { printf("%d ", i); // Output: 0 1 2 3 4 } // While Loop int j = 0; while (j < 5) { printf("%d ", j); j++; } // Do-While Loop int k = 0; do { printf("%d ", k); k++; } while (k < 5);

📊 Array dan String

Apa itu Array?

Array adalah kumpulan data dengan tipe yang sama yang disimpan berurutan di memori. Indeks array dimulai dari 0!

int angka[5] = {10, 20, 30, 40, 50}; // Mengakses elemen printf("%d", angka[0]); // Output: 10 printf("%d", angka[2]); // Output: 30 // Mengubah nilai angka[1] = 100; // Sekarang angka[1] = 100

String dalam C

C tidak memiliki tipe data string khusus. String adalah array dari char yang diakhiri dengan karakter null (\0).

char nama[] = "Budi"; // Di memori: ['B', 'u', 'd', 'i', '\0'] // Fungsi string (perlu #include <string.h>) strlen(nama); // Panjang string strcpy(nama, "Andi"); // Copy string strcmp(nama, "Andi"); // Compare string

🎯 Pointer (Konsep Penting!)

⚠️ Pointer adalah konsep tersulit tapi paling penting di C!
Pointer adalah variabel yang menyimpan alamat memori dari variabel lain.

Simbol Pointer

Simbol Nama Fungsi Contoh
& Address-of Mengambil alamat memori &x
* Dereference Mengakses nilai di alamat *ptr

Contoh Dasar Pointer

int x = 10; int *ptr = &x; // ptr menyimpan alamat x printf("%d", x); // Output: 10 (nilai) printf("%p", ptr); // Output: 0x7ffee... (alamat) printf("%d", *ptr); // Output: 10 (nilai di alamat ptr) // Mengubah nilai x melalui pointer *ptr = 20; // Sekarang x = 20

Analogi Sederhana

🏠 Analogi Rumah:

  • Variabel Biasa = Isi rumah (misal: ada TV di dalam)
  • Pointer = Alamat rumah (misal: Jl. Mawar No. 5)
  • & (Address) = Menanyakan alamat rumah
  • * (Dereference) = Masuk ke rumah dan melihat isinya

⚙️ Fungsi dalam C

Apa itu Fungsi?

Fungsi adalah blok kode yang melakukan tugas tertentu dan dapat digunakan kembali. Fungsi membantu membuat kode lebih terorganisir dan mudah dipelihara.

Struktur Fungsi

// Deklarasi (Prototipe) int tambah(int a, int b); int main() { int hasil = tambah(5, 3); printf("%d", hasil); // Output: 8 return 0; } // Definisi int tambah(int a, int b) { return a + b; }

Pass by Value vs Pass by Reference

Tipe Cara Kerja Perubahan di Fungsi
Pass by Value Salinan nilai dikirim Tidak mempengaruhi variabel asli
Pass by Reference Alamat memori dikirim Mempengaruhi variabel asli
// Pass by Value void ubahNilai(int x) { x = 100; // Tidak mengubah variabel asli } // Pass by Reference (dengan Pointer) void ubahAlamat(int *x) { *x = 100; // Mengubah variabel asli }
