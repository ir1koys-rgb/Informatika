Konsep Dasar Algoritma

Apa itu Algoritma?
Algoritma adalah urutan langkah-langkah logis dan sistematis untuk menyelesaikan suatu masalah. Seperti resep masakan: ada bahan, langkah, dan hasil akhir yang diharapkan.

Karakteristik Algoritma yang Baik

  • Input: Memiliki data masukan
  • Proses: Langkah-langkah jelas dan terurut
  • Output: Menghasilkan keluaran yang diharapkan
  • Definiteness: Setiap langkah tidak ambigu
  • Finiteness: Berakhir setelah sejumlah langkah

Contoh Algoritma Sederhana

Masalah: Menentukan apakah sebuah bilangan genap atau ganjil

// Algoritma: Cek Bilangan Genap/Ganjil START INPUT angka IF angka MOD 2 = 0 THEN PRINT "Genap" ELSE PRINT "Ganjil" END IF END
Tips Belajar: Mulailah dengan masalah sehari-hari! Coba buat algoritma untuk "membuat teh" atau "berangkat ke sekolah". Ini membantu memahami konsep sebelum masuk ke kode.

Pseudocode Dasar

Apa itu Pseudocode?
Pseudocode adalah cara menuliskan algoritma menggunakan bahasa yang mirip kode program, tetapi lebih mudah dibaca manusia. Tidak terikat sintaks bahasa pemrograman tertentu.

Struktur Dasar Pseudocode

// 1. Deklarasi DECLARE nama_variabel AS tipe_data // 2. Input INPUT nama_variabel // 3. Proses SET hasil = nilai1 + nilai2 // 4. Output PRINT "Hasil: ", hasil

Perbandingan: Algoritma vs Pseudocode vs Kode

Algoritma Pseudocode Kode Python
1. Mulai
2. Input A, B
3. Hitung C = A+B
4. Tampilkan C
5. Selesai		 INPUT A, B
SET C = A + B
PRINT C		 a = int(input())
b = int(input())
c = a + b
print(c)

Coba Tulis Pseudocode!

Tugas: Buat pseudocode untuk menghitung luas persegi panjang.

Best Practice: Gunakan kata kunci konsisten (INPUT, OUTPUT, IF, THEN, ELSE, WHILE, FOR). Indentasi membantu membaca struktur logika, meski pseudocode tidak mewajibkannya.

Pengenalan Variabel

Apa itu Variabel?
Variabel adalah "wadah" untuk menyimpan data dalam program. Seperti kotak berlabel yang bisa kita isi, ubah, dan ambil nilainya kapan saja.

Aturan Penamaan Variabel

  • ✅ Harus dimulai dengan huruf atau underscore (_)
  • ✅ Boleh mengandung huruf, angka, dan underscore
  • ✅ Case-sensitive: nama ≠ Nama
  • ❌ Tidak boleh pakai spasi atau simbol khusus (@, #, $)
  • ❌ Tidak boleh menggunakan reserved words seperti if, for, while

Tipe Data Dasar

Integer
Angka bulat
umur = 17
Float
Angka desimal
tinggi = 165.5
String
Teks/kata
nama = "Budi"
Boolean
True / False
lulus = true

Logika Dasar Komputer

Logika Boolean
Komputer bekerja dengan logika biner: TRUE (1) atau FALSE (0). Operator logika membantu menggabungkan kondisi untuk pengambilan keputusan.

Operator Logika Utama

Operator Simbol Contoh Hasil
AND && atau AND nilai >= 75 AND hadir = true True jika keduanya benar
OR || atau OR tugas_selesai OR ujian_lulus True jika salah satu benar
NOT ! atau NOT NOT (nilai < 60) Membalik nilai (True → False)

Struktur Percabangan Sederhana

// Contoh: Menentukan kategori nilai INPUT nilai IF nilai >= 90 THEN PRINT "Sangat Baik" ELSE IF nilai >= 75 THEN PRINT "Baik" ELSE IF nilai >= 60 THEN PRINT "Cukup" ELSE PRINT "Perlu Belajar Lagi" END IF
