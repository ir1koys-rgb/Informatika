Apa Itu Algoritma?

Algoritma adalah urutan langkah-langkah logis dan sistematis untuk menyelesaikan suatu masalah atau mencapai tujuan tertentu.

Analogi Sehari-hari

Bayangkan Anda ingin membuat mie instan:

  1. Rebus air hingga mendidih
  2. Masukkan mie ke dalam air
  3. Tunggu 3 menit
  4. Tiriskan mie
  5. Campur dengan bumbu
  6. Mie siap disajikan 🍜

Urutan langkah di atas adalah algoritma! Jika langkahnya diubah (misal: bumbu dicampur sebelum mie matang), hasilnya tidak akan optimal.

Sejarah Singkat

Istilah "algoritma" berasal dari nama ilmuwan Persia, Al-Khwarizmi (abad ke-9), yang memperkenalkan konsep perhitungan sistematis. Kini, algoritma menjadi fondasi seluruh pemrograman komputer.

5 Karakteristik Algoritma (Finity)

Sebuah algoritma yang baik harus memenuhi kriteria berikut:

1
Input (Masukan)

Algoritma dapat memiliki nol atau lebih masukan dari luar.

2
Output (Keluaran)

Minimal menghasilkan satu keluaran sebagai solusi.

3
Definiteness (Jelas)

Setiap langkah harus jelas, tidak ambigu, dan dapat dieksekusi.

4
Finiteness (Terbatas)

Algoritma harus berhenti setelah sejumlah langkah terbatas.

5
Effectiveness (Efektif)

Langkah-langkah harus sederhana dan dapat dikerjakan dalam waktu wajar.

Cara Menulis Algoritma

Ada dua notasi utama untuk merepresentasikan algoritma:

🔹 1. Flowchart (Diagram Alir)

Visualisasi algoritma menggunakan simbol-simbol grafis:

🔵 Mulai
📥 Input: angka A, B
➕ Hitung: Hasil = A + B
📤 Output: Hasil
🔴 Selesai

Contoh: Flowchart penjumlahan dua bilangan

🔹 2. Pseudocode (Kode Semu)

Penulisan algoritma dengan gaya mirip bahasa pemrograman, namun lebih mudah dipahami manusia:

// Algoritma: Menjumlahkan Dua Bilangan
ALGORITMA JumlahDuaBilangan
DEKLARASI
  A, B, Hasil : INTEGER

DESKRIPSI
  ← Input
  INPUT(A)
  INPUT(B)

  ← Proses
  Hasil ← A + B

  ← Output
  OUTPUT("Hasil penjumlahan: ", Hasil)
END ALGORITMA

Tips Menulis Pseudocode

  • Gunakan kata kunci konsisten: INPUT, OUTPUT, IF, WHILE
  • Tulis satu perintah per baris
  • Beri komentar (//) untuk penjelasan
  • Gunakan indentasi untuk struktur bersarang

Uji Pemahamanmu!

1. Manakah yang BUKAN karakteristik algoritma?
2. Simbel belah ketupat dalam flowchart digunakan untuk...
3. Ubah ke pseudocode: "Jika nilai ≥ 75, maka lulus; jika tidak, maka remedial"
// Jawabanmu di sini...
IF nilai >= 75 THEN
  OUTPUT("Lulus")
ELSE
  OUTPUT("Remedial")
ENDIF
Gunakan struktur IF-THEN-ELSE. Pastikan kondisi menggunakan operator perbandingan (>=, <, dll).
